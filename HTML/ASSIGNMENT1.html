Study of Web Technology Basics

1. HTML (HyperText Markup Language)

HTML is the standard markup language used to create web pages. It structures the content using tags such as <h1>, <p>, <div>, etc.

2. CSS (Cascading Style Sheets)

CSS is used to define the style of a web page including layout, colors, and fonts. It can be included within HTML or as an external file.

3. JavaScript

JavaScript is a scripting language used to add interactivity to web pages such as validating forms, creating dynamic content, etc.

4. Backend Web Technologies

Backend languages are used to handle the server-side logic of web applications:

  1. PHP – A server-side scripting language mainly used for web development.
  2. Python – Used in web frameworks like Django and Flask.
  3. Node.js – Allows JavaScript to run on the server side.

5. Database Technologies

Databases are used to store and manage data for web applications.

  • MySQL
  • MongoDB
  • PostgreSQL

Conclusion

Web technologies include a combination of frontend (HTML, CSS, JavaScript) and backend (PHP, Python, Node.js) tools that together create full-featured web applications.
